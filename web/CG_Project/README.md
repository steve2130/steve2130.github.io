# COMP4422 Computer Graphics - Project
## Group 4

1. Make sure you have Python installed (https://www.python.org/downloads/)  
  
2. Open Powershell/CMD in the root folder (Type [Powershell] or [CMD] in the address bar of the explorer)  
  
3. In Powershell/CMD, type [python server.py]. A web server will be hosted locally.  
  
4. Enter [http://localhost:8000/Code/threejs.html] in any modern web browser  

  ---
No external dependencies required for the python file  
Only Python standard library modules is used:  
 - http.server  
 - socketserver  
 - mimetypes  
 - os  
  
Python 3.6+ is recommended
